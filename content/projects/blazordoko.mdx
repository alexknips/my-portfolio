This application demonstrates the use of SignalR to create a Doppelkopf Blazor application.

Blazor is a web framework developed by Microsoft that allows you to build interactive web applications using C# instead of JavaScript. With Blazor, you can write your application logic and UI components in C#, which is then executed on the server or in the browser using WebAssembly.
